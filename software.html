Linux - Unix like operating system, lightweight and flexible

Linux is a Unix-like operating system that was designed to provide an operating system comparable to traditional Unix systems. Linux systems are efficient, flexible and fast-performing based upon the Linux kernel developed by Linus Torvalds.

The operating system can optionally include a graphical user interface, an X Window System, TCP/IP, the Emacs editor, as well as other components found in Unix systems. Linux is distributed using the Free Software Foundation's copyleft stipulations that mean any modified version that is redistributed must in turn be freely available.
